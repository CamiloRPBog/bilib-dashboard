Conexión con Google Sheets

Configura la sincronización bidireccional con tu hoja de cálculo

Paso 1 — Instalar el script en Google Sheets

1. Abre tu Google Sheet → Menú Extensiones → Apps Script
2. Borra el contenido y pega el archivo bilib2026_apps_script.js que descargaste
3. Guarda (Ctrl+S) → Renombra el proyecto como BILIB2026
4. Clic en Implementar → Nueva implementación
5. Tipo: Aplicación web · Ejecutar como: Yo · Acceso: Cualquier persona
6. Clic Implementar → Autoriza los permisos → Copia la URL generada

Paso 2 — Pega la URL del Web App aquí

La URL tiene este formato: https://script.google.com/macros/s/AKfyc.../exec

Paso 3 — Estructura de columnas requerida en el Sheet

La fila 1 debe tener exactamente estos encabezados (o puedes ejecutar replaceAll desde el botón de abajo para que se creen automáticamente):

A: IDB: TipoC: NombreD: Descripcion E: TrimestreF: ColorG: OwnerH: KR_ObjID I–M: KR datosN–R: Tarea datosS–U: Miembro datos
único objetivo / kr / tarea / miembro texto texto Q1-Q4 #hex nombre ID del obj padre start, target, current, unit, status asignado, fecha, prioridad, estado, objId rol, email, color

Sincronización manual

El auto-sync lee el Sheet cada 30 segundos y actualiza el dashboard automáticamente.
Cualquier cambio que hagas en el dashboard también se escribe de vuelta al Sheet en tiempo real.
