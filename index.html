Google Sheets Connection

Configure two-way sync with your spreadsheet

Step 1 — Install the script in Google Sheets

1. Open your Google Sheet → Menu Extensions → Apps Script
2. Delete the content and paste the file bilib2026_apps_script.js que descargaste
3. Save (Ctrl+S) → Rename the project to BILIB2026
4. Click on Deploy → New deployment
5. Type: Web app · Run as: Yo · Access: Anyone
6. Clic Deploy → Authorize permissions → Copy the generated URL

Step 2 — Paste the Web App URL here

The URL looks like this: https://script.google.com/macros/s/AKfyc.../exec

Step 3 — Required column structure in the Sheet

Row 1 must have exactly these headers (o puedes ejecutar replaceAll desde el botón de abajo para que se creen automáticamente):

A: IDB: TipoC: NameD: Descripcion E: QuarterF: ColorG: OwnerH: KR_ObjID I–M: KR datosN–R: Task datosS–U: Miembro datos
único objetivo / kr / tarea / miembro texto texto Q1-Q4 #hex nombre ID del obj padre start, target, current, unit, status asignado, fecha, prioridad, estado, objId rol, email, color
